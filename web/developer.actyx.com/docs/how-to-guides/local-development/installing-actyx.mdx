## Hardware Requirements

  groupId="os"
  defaultValue="android"
  values={[

    { label: 'Windows', value: 'windows' },
    { label: 'Linux', value: 'linux' },

    { label: 'Docker', value: 'docker' },
  ]}
>
<TabItem value="android">


In order to be able to run Actyx on Android, your device needs to meet the following requirements:

- Android 6+
- At least 2GB of RAM
- `x86`, `arm64-v8a` or `armeabi-v7a` [ABI](https://developer.android.com/ndk/guides/abis.html#sa)

You can find a list of devices that we know to work well [here](/faq/supported-edge-devices.md).

</TabItem>
<TabItem value="windows">


In order to be able to run Actyx on Windows, your device needs to meet the following requirements:

- Windows 10 or Windows Server 2019 / 2016
- amd64 processor architecture

</TabItem>
<TabItem value="linux">


In order to be able to run Actyx on Linux, your device needs to meet the following requirements:

- `amd64`, `aarch64`, `armv7` or `arm` processor architectures

</TabItem>
<TabItem value="mac">


Your device must run a recent version of macOS (> 10.14 Mojave) to be able to run Actyx guaranteed, though earlier macOS versions may work as well.
New Apple devices with the ARM-based Apple M1 chip will use Rosetta to run the binary (an ARM build will be released in the future).

</TabItem>
<TabItem value="docker">


In order to be able to run Actyx on Docker, your device needs to meet the following requirements:

- Must have Docker installed
- Must be able to run Docker in privileged mode
- Must be able to run amd64 or arm64v8 Docker images
- You can find a list of devices that we know to work well here.

:::info
Note that Actyx on Docker should only be used if there is no other way, as the Docker architecture poses problems with peer discovery mechanisms.
:::

</TabItem>
</Tabs>


## Installing Actyx

<Tabs
  groupId="os"
  defaultValue="android"
  values={[
    { label: 'Android', value: 'android' },
    { label: 'Windows', value: 'windows' },
    { label: 'Linux', value: 'linux' },
    { label: 'macOS', value: 'mac' },
    { label: 'Docker', value: 'docker' },
  ]}
>
<TabItem value="android">


Actyx is publicly available for Android in the [Google Play Store](https://play.google.com/store/apps/details?id=com.actyx.os.android).
Just open the Google Play Store on your device, search for Actyx and install it.
In case you don't have access to the Play Store, please visit [our downloads page](https://www.downloads.actyx.com) and download it from there.
Once downloaded, simply follow the instructions on your device to install Actyx.
After clicking on the Actyx icon on your home screen, Actyx starts.

</TabItem>


<TabItem value="windows">


You can download the Actyx installer for Windows on our [downloads page](https://www.downloads.actyx.com).
After opening the installer, you are guided through the setup process.
During this process, you can choose to install all or a subset of the following tools:

- Actyx
- Actyx CLI
- Actyx Node Manager

To start Actyx, simply press the Windows key on your keyboard, search for Actyx and hit enter.
You will now see an Actyx tray icon in your taskbar, indicating that Actyx is running.

</TabItem>


<TabItem value="linux">


You can download the binary for ActyxOS on Linux on [downloads page](https://www.downloads.actyx.com).
For Actyx to start, please open your terminal, navigate to the directory where you saved the binary and type in the following:

```
chmod +x actyx
```

Now, you can simply run the binary with

```
./actyx
```

</TabItem>


<TabItem value="mac">


You can download the binary for ActyxOS on macOS on the [downloads page](https://www.downloads.actyx.com).
For Actyx to start, please open your terminal, navigate to the directory where you saved the binary and type in the following:

```
chmod +x actyx
```

Now, you can simply run the binary with

```
./actyx
```

</TabItem>

<TabItem value="android">


While Actyx is running, simply pull down from the top of your screen.
You should now see that Actyx is running in your notifications overview.
Please note that Actyx keeps running when you swipe the window away in the app switcher.
In order to properly stop Actyx, please click the button in the UI or force stop it in the system settings.
