Actyx provides libraries for developing TypeScript/JavaScript application: Actyx SDK and MachineRunner.
Actyx SDK simplifies the way application connects, and publish and query events.
Meanwhile, MachineRunner allows modeling computation as operation-based distributed state machine where operations are stored as events in Actyx.

The libraries are available as npm packages:
