## ax nodes ls

This command returns the current state of the specified node in form of a table.

### Usage

`ax nodes ls [FLAGS] <NODE>...`

### Flags

<CLIRef code={'`-h, --help`'} description={`Prints help information`} />
<CLIRef code={'`-V, --version`'} description={`Prints version information`} />
<CLIRef code={'`-l, --local`'} description={`Process over local network`} />
<CLIRef
  code={'`-v`'}
  description={`Verbosity level. Add more v for higher verbosity (-v, -vv, -vvv, etc.)`}
/>

### Arguments

<CLIRef
  code={'`<NODE>...`'}
  description={`Node ID or, if using \`--local\`, the IP address of the node to perform the operation on`}
/>

<Divider />

<Callout
  icon="⚠️"
  headline="Output of `ax nodes ls`"
  body="If the node is reachable, the output of \`ax nodes ls\` will show you its status. If the node is unreachable, the output contains information why the node could not be reached. The Actyx CLI distinguishes 2 cases:
- Host unreachable
- ActyxOS unreachable (this means the host was reachable but the TCP connection reset)"
/>

### Examples

Get the status of all specified nodes in the local network:

```text
$ ax nodes ls --local 10.2.3.23 10.2.3.24
```

```text
+------------+--------------+---------+----------+---------+---------------------------+---------+
| NODE ID    | DISPLAY NAME | STATE   | SETTINGS | LICENSE | STARTED                   | VERSION |
+------------+--------------+---------+----------+---------+---------------------------+---------+
| 10.2.3.23  | MY NODE      | running | valid    | valid   | 2020-08-31T09:00:00+00:00 | 1.0.0   |
+------------+--------------+---------+----------+---------+---------------------------+---------+
```

Get the status of all specified nodes in the local network as a json object

```text
$ ax --json nodes ls --local 10.2.3.23 10.2.3.24 10.2.3.25
```

```json
{
  "code": "OK",
  "result": [
    {
      "connection": "reachable",
      "nodeId": "10.2.3.23",
      "displayName": "MY NODE",
      "state": "running",
      "settingsValid": true,
      "licensed": true,
      "appsDeployed": 23,
      "appsRunning": 17,
      "startedIso": "2020-05-19T07:52:26+00:00",
      "startedUnix": 1589874746,
      "version": "1.0.0"
    },
    {
      "connection": "actyxosUnreachable",
      "host": "10.2.3.24"
    },
    {
      "connection": "hostUnreachable",
      "host": "10.2.3.25"
    }
  ]
}
```
