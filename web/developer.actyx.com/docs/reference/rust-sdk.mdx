Building apps in [Rust](https://www.rust-lang.org/) and want to easily create and access data streams in your Actyx swarm? That's what we built the Actyx SDK for Rust for. The [`actyx_sdk` crate](https://crates.io/crates/actyx_sdk) defines all necessary data types and provides Rust bindings for communicating with the Actyx [Events API](../reference/events-api.mdx).

## Installation

Just run `cargo add actyx_sdk` (requires [`cargo-edit`](https://crates.io/crates/cargo-edit)), or add the following dependency manually to your `Cargo.toml` file:

```yml
[dependencies]
actyx_sdk = "0.2.0"
```

## Documentation

You can find the latest documentation for the Rust SDK on [docs.rs](https://docs.rs/actyx_sdk/0.2.0/actyx_sdk/).
