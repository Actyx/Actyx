Building apps in [Rust](https://www.rust-lang.org/) and want to easily create and access data streams in your ActyxOS swarm? That's what we built the ActyxOS SDK for Rust for. The [`actyxos_sdk` crate](https://crates.io/crates/actyxos_sdk) defines all necessary data types and provides Rust bindings for communicating with ActyxOS's [Event Service API](/docs/reference/event-service.mdx).

## Installation

Add the following dependency to your `Cargo.toml` file:

```yml
[dependencies]
actyxos_sdk = "~0.2"
```

## Documentation

You can find the latest documentation for the Rust SDK [on docs.rs](https://docs.rs/actyxos_sdk).
